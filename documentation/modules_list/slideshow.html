Frontend Appearance

Create new slide

This allows you to to create new slide item.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Pos Slideshow Slider module, click "Configure" button.

  5. In part SLIDES LIST, click ADD NEW

  6. Add the info for the new slide.

    We got:

    • Slide name: Name of slide item
    • Position slide: Choose position of this slide
    • Select a file: Click on Choose a file to upload slide image
    • Target URL: Set a url for the targeted slide (Target URL is only used for buttons)
    • Caption: Click Add button to add description for this slide

      - ENGLISH (ENGLISH)/FRANÇAIS (FRENCH): Choose language for title, text or description

      - Name: Add title, text or description

      - Group display: Choose the style for for title, text or description

      - Animation: Choose text animations

    • Enable: Choose Yes/No to enable the slide
  7. Click Save button when you finish

Configure Slideshow

General Configuration

This allows you to to config selected elements as general configuration for the Slideshow.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Pos Slideshow Slider module, click "Configure" button.

  5. In part SETTINGS

    We got:

    • Speed: Set the speed of an animation effect of text description
    • Delay time: Set time duration for moving to next slide
    • Pause on hover: Select Yes to pause when the mouse is over the slideshow
    • Show Caption: Select Yes/No to show/hide caption
    • Show navigation: Select Yes/No to show/hide navigation
    • Show pagination: Select Yes/No to show/hide pagination

    => Configure parameters as you want

  6. Click Save button when you finish

Config Slide List

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Pos Slideshow Slider module, click "Configure" button.

  5. In part SLIDES LIST, click Edit on selected slide item

    => Configure parameters as you want

  6. Click Save button when you finish

Note: In part SLIDES LIST, using drag / drop function to move slide item into another position.

Translate module

  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Installed modules” tab
  • On Pos Slideshow Slider module, click "Configure" button.
  • In part SLIDES LIST, click Edit on selected slide item
  • Click to choose language you want to translate.
  • At Caption, choose tab another language (your language) to translate description of slide.
  • Click Save button when you finish
