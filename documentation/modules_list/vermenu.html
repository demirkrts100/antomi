Create new menu item

General Setting

This allows you to to set selected elements for the Vertical MegaMenu.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Vertical Megamenu module, click "Configure" button.

  5. In part MENU SETTINGS

    We got:

    • Heading title: Add heading title
    • Background type: Select background type

      - None: Don't use background type

      - Color

      • Background color: Set color for background

      - Image

      • Image source: Upload image background from computer
      • Background repeat: Background image can be repeated along the horizontal and vertical axes, or not repeated at all
    • Submenu animation: Select animations for dropown menu items
    • Custom CSS: Add custom CSS for menu item
    • Catalog number displayed: Specify the number of catalogs on vertical mega menu
  6. Click Save button when you finish

How to build a simple menu item?

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Vertical Megamenu module, click "Configure" button.

  5. Click the Add new item then add the info for the menu item.

    We got:

    • Title: Add menu title (>>> If use Prestashop link and want to use Prestashop title, leave it empty.)
    • Type Link: Select menu item type (PrestaShop Link / Custom Link / None)

      - PrestaShop Link: Select link from Category, CMS, Supplier, Manufacturer or Page.

      - Custom Link: Enter internal or external link

    • Sub Title: Enter sub title
    • Use icon: Choose icon image

      - No: Don't use icon

      - Font-Awesome Icon: Use font-awesome icon (ex: <i class="fab fa-angellist"></i>)

      - Image Icon: Click Select image > to upload icon from computer

    • Main link style: Config CSS for menu item (ex: color, color hover, background color...)
    • Subtitle style: Config CSS for sub title (ex: color, background color, font size...)
    • Submenu type: There are 3 types of submenus: Mega, Flyout and None (Flyout only is available when you select category in Type Link)
    • Add specific class: Class to style the menu item
    • New window: Choose Yes/No to open new window/tab or not by clicking
    • Active: Choose Yes pulish the menu item
  6. Click Save button when you finish

How to build columns in Sub-menus?

In here, you can set the width of the sub-megamenu, add column, and add class that style for it. To use this setting, select the menu item as Mega type in Submenu type field.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Vertical Megamenu module, click "Configure" button.

  5. Click the Build SubMenu then add the info for the menu item.

  6. Click on Add a row and then click Save to create new row

  7. Click on Add Column to add column for this row

    We got:

    • Width: Configure column-width. The maximum of columns is 12 based on the bootstrap columns format.
    • Column title: Enter title of column (>>> If use Prestashop link and want to use Prestashop title, leave it empty.)
    • Type Link: Select type for column title(PrestaShop Link / Custom Link / None)

      - PrestaShop Link: Select link from Category, CMS, Supplier, Manufacturer or Page.

      - Custom Link: Enter internal or external link

    • Specific class: Class to style the column title
    • Active in mobile: Choose Yes pulish in the mobile

  8. Click Save button when you finish

  9. After that, you need configure submenu item

    We got:

    • Type item: Select submenu item type (Category tree, Prestashop Link, Custom Link, Product, Banner image, HTML block, Manufacturer logo)

      - Category tree

      • Select a category: Select category

      - Prestashop Link

      • Prestashop link: Select link from Category, CMS, Supplier, Manufacturer or Page.

      - Custom Link

      • Title: Enter custom block title
      • URL: Enter internal or external link

      - Product

      • Product: Choose product

      - Banner image

      • + Image source: Upload banner image from computer
      • + Image link: Enter internal or external link for banner image

      - HTML block

      • HTML content: Enter the HTML to define the column

      - Manufacturer logo

      • Select manufacturer: Select manufacturer logo from list
    • Active in mobile: Choose Yes pulish in the mobile

Configure Vertical Megamenu

General Configuration

This allows you to to config selected elements as general configuration for the Vertical MegaMenu.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Vertical Megamenu module, click "Configure" button.

  5. In part MENU SETTINGS

    => Configure parameters as you want

  6. Click Save button when you finish

Config menu item

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Vertical Megamenu module, click "Configure" button.

  5. In part MENU LIST, click Edit on selected menu item

    => Configure parameters as you want

  6. Click Save button when you finish

Note: In part Menu List, using drag / drop function to move menu item into another position.

Config Submenu

Configure the width of the sub-megamenu, change title, link and add class that style for submenu.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Vertical Megamenu module, click "Configure" button.

  5. Click the Build SubMenu

    => Configure parameters as you want (edit, add new or delete row, column; add new submenu item)

Note: In this part, using drag / drop function to move row in a single column, or the columns with each other.

Translate module

Translate Title

  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Installed modules” tab
  • On Vertical Megamenu module, click "Configure" button.
  • In part MENU LIST, click Edit on selected menu item
  • Click to choose language you want to translate.
  • At Title, translate the menu item.
  • Click Save button when you finish

Translate PrestaShop Link

1. Category Catalog

  • Login your Prestashop admin
  • Go to Catalog > Catagories
  • On selected catagory, click "Edit" button.
  • Click to choose language you want to translate.
  • At Name, enter your language & click Save button when you finish

2. Product Catalog

  • Login your Prestashop admin
  • Go to Catalog > Products
  • On selected catagory, click "Edit" button.
  • Click to choose language you want to translate.
  • At Name, enter your language & click Save button when you finish

3. Cms

  • Login your Prestashop admin
  • Go to Design > Pages
  • On selected page, click "Edit" button.
  • Click to choose language you want to translate.
  • At Title, enter your language & click Save button when you finish

4. Manufacturer/Supplier

  • Login your Prestashop admin
  • Go to Catalog > Brands & Supplier
  • Click on BRANDS/SUPPLIER tab
  • On selected Manufacturer/Supplier, click "Edit" button.
  • Click to choose language you want to translate.
  • At Title, enter your language & click Save button when you finish

5. Page

  • Login your Prestashop admin
  • Go to Shop Parameters > Traffic & SEO
  • Click on SEO & URLS tab
  • At SEO & URLs part, click Edit on selected page.
  • Click to choose language you want to translate.
  • At Page title , enter your language & click Save button when you finish
