How to add new static block

  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Installed modules” tab
  • On Pos Static Footer module, click "Configure" button.
  • Click the New block button then add the info for the new static block.

  • We got:

    • Name of the block: Add staticblock title
    • Show title: Choose Yes/No to show title or not
    • Width of this block: Configure column-width. The maximum of columns is 12 based on the bootstrap columns format.
    • Hook: Select hook for this static block
    • Select type content:

      - List Links: Select link or add external link as you need

      - Custom Content: Custom static blocks or select module to display on homepage.

  • Click Save button when you finish

Note: Theme's hooks position in Front-office

Configure Pos Static Footer module

Configure static blocks in the Pos Static Footer module and add them to any place on the page.

  1. Login your Prestashop admin

  2. Go to Modules > Modules Manager

  3. Choose “Installed modules” tab

  4. On Pos Static Footer module, click "Configure" button.

  5. Click Edit on selected static footer and then configure parameters as you want

  6. Click Save button when you finish

Note: With drag and drop, you can rearrange the structure and elements at the footer

Translate module

Translate Title

  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Installed modules” tab
  • On Pos Static Footer module, click "Configure" button.
  • Click Edit on selected static block and then click to choose language you want to translate.
  • At Name of the block, HTML content, translate text of this staticblock.
  • Click Save button when you finish

Translate List Link

  • Login your Prestashop admin
  • Go to Shop Parameters > Traffic & SEO
  • Click on SEO & URLS tab
  • At SEO & URLs part, click Edit on selected page.
  • Click to choose language you want to translate.
  • At Page title , enter your language & click Save button when you finish
