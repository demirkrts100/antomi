Configure Pos Countdown time price

This allows you to to config selected elements as general configuration for Pos Countdown time price.

  1. Login your Prestashop admin
  2. Go to Modules > Modules Manager
  4. Choose “Installed modules” tab
  6. On Pos Countdown time price” module, choose “Disable” if you don’t want to show countdown time.
  8. Click Save button when you finish

How to set time for discount product

This allows you to to config selected elements as general configuration for Pos Countdown time price.

  • You set product as discount product. Please see in Pos Special Products module
  • From the Specific Prices tab, when you Add a new specific price, you choose discount prior and set discount:
    - from: Select a beginning date for this discount.
    - to: Select an expiration date for this discount.
  • Click "Save" button

Translate module

  • Login your Prestashop admin
  • Go to Modules > Modules Manager
  • Choose “Installed modules” tab
  • On Pos Countdown time price module, click "Configure" button.
  • Click to choose language you want to translate.
  • At Title, translate the menu item.
  • Click Save button when you finish
